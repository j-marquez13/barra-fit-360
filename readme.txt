Módulo 3: Punto de Venta (POS) MultimonedaAquí es donde registras las ventas del día.Tabla Ventas: id, id_usuario, fecha, metodo_pago (Binance, Zelle, Bancolombia, Efectivo), moneda_original, tasa_aplicada, total_cop.Función de Conversión: El sistema debe tener un campo de entrada para la "Tasa del día".Fórmula: Monto_Recibido_Bs / Tasa_Bs_COP = Total_COP. El sistema siempre debe guardar el registro en COP para mantener la contabilidad uniforme.Módulo 4: Cuentas por CobrarPara integrar el archivo "CUENTAS POR COBRAR JUNIO 2026.xlsx".Tabla Clientes_Credito: id, nombre, limite_credito, saldo_actual.Tabla Movimientos_Credito: id, id_cliente, tipo (Cargo/Abono), monto, fecha, nota.Módulo 5: Motor de Cierres (Reportes Automáticos)Para replicar tus archivos de "Cierre Diario" y "Cierre Semana".Procedimiento de Cierre: Al finalizar el turno, el sistema debe ejecutar una consulta SQL que:Filtre todas las ventas donde id_usuario sea del turno actual.Agrupe los montos totales por metodo_pago.Reste el costo_produccion total de los productos vendidos (calculado mediante la suma de insumos) para mostrar la Utilidad Neta.Genere una tabla visual idéntica a tus Excel de cierre.Instrucción para Antigravity (Copia y pega esto)"Actúa como mi Arquitecto de Software Senior.
